# Habit Tracker API
API RESTful para rastrear hábitos pessoais, permitindo que usuários:
- Cadastrar hábitos diários (exercício, leitura, meditação)
- Registrar check-ins diários
- Visualizar streaks (sequências de dias consecutivos)
- Estatísticas e gráficos de progresso

---

### Tecnologias

- **Framework**: FastAPI
- **Banco de Dados**: PostgreSQL Assíncrono
- **ORM**: Tortoise ORM
- **Validação**: Pydantic
- **Migrações**: Aerich
- **Conteinerização**: Docker
- **Linters/Formatters/Tasks**: Black, Isort, Taskipy
